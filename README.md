# Security Labs

This repository documents hands-on cybersecurity labs focused on blue team
operations, detection engineering, and incident response.

## Background
I am a Cyber Security graduate with experience in technical operations,
automation, and data analysis. This repository demonstrates practical security
skills through realistic lab environments and documented investigations.

## Focus Areas
- Active Directory security
- Log analysis and SIEM detection
- Windows and Linux logging
- Incident investigation and response

## Tools (Planned)
- Windows Server (Active Directory)
- Windows 10/11 clients
- Linux (Ubuntu)
- SIEM (Wazuh / Splunk)
- Firewall (pfSense)

## Repository Structure
Each lab includes:
- Environment overview
- Architecture diagram
- Setup notes
- Security logging configuration
- Observations and lessons learned
